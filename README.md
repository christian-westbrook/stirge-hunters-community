# stirge-hunters-community

This repository contains the community edition of Stirge Engine, a tool for building RSClike MMOs.

[Stirge Hunters](https://stirgehunters.com) is an indie RSClike MMO being built in public by [a solo developer](https://www.linkedin.com/in/christian-westbrook/). The `stirge-hunters-community` repository makes some of the codebase powering Stirge Hunters available for other developers to use in building their own RSClike MMOs.

Stirge Engine is comprised of a collection of systems that work together to deliver an MMO. This collection includes:
- Conduit, a browser-based MMO client
- Crucible, a tick-based MMO server
- Crier, a web system for hosting and promoting a browser-based MMO

Note that I don't have complete versions of each of these tools in place in the community repository yet. You can expect this repository to start small and slowly expand as source code is extracted from Stirge Hunters.

Note also that the goal of this repository is not to serve as a general-purpose tool for building MMOs, but rather to make building a specific kind of MMO easier.

Games built with Stirge Engine can easily be deployed alongside Stirge Hunters and other MMOs on the Stirges MMO Platform, a cluster of game servers dedicated to hosting indie MMOs. Anyone interested in having their MMO hosted can reach out to twinnrova@stirgehunters.com.

## FAQ

*Will the client stay browser only?*

No, a desktop client is planned for the future. A mobile client could also be considered with enough demand for it.
